# Weather Package

## Installation
```python
pip install -i https://test.pypi.org/simple/ weather-Mike-Redfox
```

## Usage 

```python
from weather import weather
weather.weather("{name of your city}")
```
If you don't pass any arguments it defaults to London

## Example 

```python
from weather import weather

weather.weather("Madrid")
```
